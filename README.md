# SAII Application - Hajj Transportation System

## Project Description
SAII is a professional web-based management system specifically designed to optimize transportation services for pilgrims during Hajj. The application bridges the gap between pilgrims and trip organizers to ensure a smooth, organized, and time-efficient journey.

## Main Features
* Authentication: Secure User Login and Registration system.
* Trip Management: Full CRUD (Create, Read, Update, Delete) operations for trip bookings.
* Smart Features: QR Code generation for booking verification and HeatMap monitoring for crowd density.
* Admin Dashboard: Dedicated interface for administrators to manage schedules and monitor all active trips.

## Technologies Used (Frontend Phase)
* HTML5 & CSS: For structuring and styling the user interface.
* JavaScript: To handle client-side logic and dynamic UI updates.

## Project Structure
1.  Source Files: All HTML, CSS, and JS files required to run the frontend.
2.  README.md: Documentation and project overview.
3.  AUTHORS: Detailed list of contributors and team members.

## How to Run
1. Clone or download the repository.
2. Navigate to the project root directory.
3. Open index.html in any modern web browser.
